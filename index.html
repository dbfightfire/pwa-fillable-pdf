Fillable PDF Maker (PWA)

What is this? This app can create a fillable PDF (users type in the PDF later), or export a PDF pre-filled from the form below. It works offline once installed.

Tip: You can change the fields without coding—open app.js and edit the FIELDS list.

Form (HTML → PDF)

Advanced: You can also upload a PDF that already has form fields and we will try to fill matching fields.

How it works (Plain-English)

  1. Press Download empty fillable PDF → we build a new PDF with fields you can type into.
  2. Or fill the web form and press Download PDF with my answers → we put your answers into a PDF.
  3. Click Install App to use it offline like a normal app.
Made with pdf-lib in your browser. No server needed.
